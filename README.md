# HelloWorldWebCore
A basic Hello World Asp.Net application for showcasing CI/CD scenarios

* Create a Pipeline
* Connect to GitHub
* Use a Windows image or another that supports .NET Core 2.1
* Configure CodeDeploy with the image, use AS
